A) Strategic Minimalism + Broadcast

Filosofia: "Cada elemento reduz esforço mental ou sai." Zero decoracao. Tipografia bold como unica linguagem visual. Inspirado em FanDuel + ESPN broadcast.

Tecnicas 2026: Container Queries para cards adaptativos, @starting-style para entrada de picks, Server Components para SEO, View Transitions entre lutas.

Pro: Maximo foco na acao (prever), carrega rapido, SEO excelente.
Contra: Pode parecer "frio" — menos vibe UFC.

B) Atmospheric Fight Night

Filosofia: "Entrar no Arena = entrar na noite de luta." Glows ambientais sutis (nao neon gritante), gradientes que respiram, hierarquia dramatica. Energia controlada.

Tecnicas 2026: CSS @starting-style para micro-animacoes de entrada, conic-gradient para auras, Container Queries, Server Components com Suspense streaming.

Pro: Tem a vibe UFC sem ser over-the-top. Emocionante mas legivel.
Contra: Mais CSS = potencial impacto em performance (mitigavel).

C) Checklist-Driven (Hibrido)

Filosofia: "Seu card eh uma lista de tarefas." Mostra TODAS as lutas como checklist, progresso visivel, proximo passo claro. Clicou na luta → expande card de pick. Storytelling = completar o card.

Tecnicas 2026: Server Component renderiza lista, Client Component so para interacao. ISR para dados de evento. View Transitions ao expandir luta. Progressive Disclosure total.

Pro: Maximo storytelling — usuario SABE onde esta e o que falta. Zero confusao.
Contra: Menos "wow factor" visual. Mais funcional que emocional.
